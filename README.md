# 🚀 MemeForge-AI (Desktop App)

Willkommen zum offiziellen Repository von **MemeForge-AI**! Ein moderner AI-Video Meme Generator.

## 🛠️ Tech Stack
- Frontend: Electron, React, Vite

## 🚀 Lokale Entwicklung

```bash
# Installiere Abhängigkeiten
npm install

# Starte die Vite-Entwicklungsumgebung inklusive Electron
npm run electron:dev

# Baue die Produktions-Version
npm run electron:build
```
