# 🚀 MemeForge-AI Studio

Willkommen bei **MemeForge-AI**! Deine Desktop-Software, um mit der Kraft der KI unfassbar witzige Memes und Videos zu generieren. 🎨✨

## 📥 Installation (Super Einfach!)

Du musst kein Programmierer sein, um diese Software zu nutzen! Folge einfach diesen Schritten:

1. **Lade den Ordner herunter**: Lade dieses gesamte Repository auf deinen PC herunter.
2. **Doppelklick**: Führe die Datei `deploy.bat` mit einem einfachen Doppelklick aus. 🖱️
3. **Fertig!**: Das Skript installiert vollautomatisch alles, was du brauchst. Danach öffnet sich das Programm von selbst! 🎉

*(Tipp: Wenn du das Programm später wieder öffnen möchtest, nutze einfach die `launch.bat`!)*

## 💡 So funktioniert's

- **Schritt 1: Registrieren / Einloggen** 🔐
  Wenn du die App zum ersten Mal öffnest, erstelle dir schnell einen Account. Deine Daten werden sicher in der Cloud gespeichert!
- **Schritt 2: Lizenz aktivieren** 🔑
  Du startest automatisch im **FREE** Modus. Um die volle KI-Power zu nutzen, gib deinen `PRO`-Lizenzschlüssel ein. Dank **Echtzeit-WebSockets** aktualisiert sich dein Rang sofort live in der Software! ⚡
- **Schritt 3: Memes Generieren** 🎬
  Gib einfach deinen Text (Prompt) ein und klicke auf "Generieren". Die Python KI-Engine übernimmt die Arbeit und zaubert dir dein Video.

## 💎 Features
- **Premium "ZeNmOdZ" Design**: Glassmorphism, leuchtende Neon-Animationen und ein cleanes Dark Theme. 🌌
- **Live-Updates**: Lizenzen werden in Sekundenbruchteilen verarbeitet, das Rang-Badge beginnt sofort zu glühen! ⏱️
- **Cloud Accounts**: Wenn du die Software auf einem anderen Gerät öffnest, loggst du dich einfach wieder ein. Nichts geht verloren! ☁️
